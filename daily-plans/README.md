# Daily Plans (150 days)

One file per working day. Open today's file, do the work, check off tasks.

## How to use

1. **Morning:** Open `day-NNN.md` for today
2. **Work:** Follow tasks, check resources
3. **Evening:** Tick off completed items
4. **Night:** Open tomorrow's file, pre-decide first task (5 min)
5. **End:** Fill `tracker/days/day-NNN.md` log

## Structure

Each daily file has:
- **Today:** specific tasks (from CURRICULUM.md)
- **Resources:** links needed
- **Deliverable:** what ships (if applicable)
- **Cadences:** ongoing commitments (LinkedIn, X, LeetCode, Sunday block)
- **Done?:** end-of-day checklist
