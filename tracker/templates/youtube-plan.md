## Video production workflow (1 hr per video)

### Pre (10 min)
- [ ] Open template: `notes/youtube/episode-NNN.md`
- [ ] Outline 3-5 bullet points
- [ ] Test mic + lighting + screen recording

### Record (20-30 min)
- [ ] OBS or Loom (depending on format)
- [ ] Don't restart on mistakes — talk through them
- [ ] Aim for 1 take, raw

### Edit (10-15 min)
- [ ] Trim front + back
- [ ] If needed: cut 1-2 mistakes
- [ ] **Don't:** add intros, music, fancy transitions (time killers)

### Publish (10 min)
- [ ] Title (use formulas above)
- [ ] Thumbnail in Canva (text + face/screenshot)
- [ ] Description with: 1-line summary + chapters + GitHub repo + LinkedIn + previous video link
- [ ] Tags: AI Engineer, LLM, Python, RAG, Fintech AI, [project name]
- [ ] Pin first comment with key links
- [ ] Share to LinkedIn + X same day

### Total: 50-65 min

## When to KILL the channel (be honest with yourself)

If by Day 90:
- Less than 50 subs
- Fewer than 100 avg views per video
- Channel takes more than 3 hrs/week consistently
- Demoralizing, not energizing
- Hurting Project 5 progress

→ **Pause channel. Focus on LinkedIn/X. Restart later if desired.**

YouTube is OPTIONAL. Don't let it kill the main mission.

## Anti-patterns

- ❌ Spending 6 hrs editing one video — kills cadence
- ❌ Trying to compete with AI Jason / James Briggs production quality
- ❌ Reaction videos to OpenAI announcements (everyone does this)
- ❌ Tutorial channel without unique angle (saturated)
- ❌ Ignoring comments (kills algorithm boost)
- ❌ Posting then dipping — engagement matters
