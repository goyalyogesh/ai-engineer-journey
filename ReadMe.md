## Strategy: 2-Stage Career Pivot (Recommended)

**Reasoning:** Skipping C#/.NET-heavy AI roles entirely is risky. With 0 AI experience today, hybrid roles hire faster. Pure Python AI roles compete against ML PhDs, bootcamp grads, and CS grads with strong Python portfolios. Better to leverage existing .NET skills as a bridge.

### Stage 1 (Months 0-6): Get AI on Resume — Take ANY AI-adjacent role
- Hybrid .NET + Azure OpenAI / Semantic Kernel
- .NET backend on team that ships AI features
- Use Python on side projects evenings/weekends
- **Goal:** "AI" on resume + paycheck continuity

### Stage 2 (Months 6-18): Pivot to Pure Python
- Now have production AI experience (rare and valuable)
- Python side projects + real AI work = strong candidate
- Jump to Python-only role with leverage and choice

### When pure-Python jump makes sense (skip Stage 1):
- 12+ months savings cushion
- Can afford 6-month unemployment
- Young, no dependents
- Portfolio already strong (3+ shipped projects)

### When 2-stage is better:
- Need income continuity
- 30+ with responsibilities
- Few AI startups in local market

## Reality Check
- .NET backend skills still help — system design, APIs, production thinking transfer
- Lead resume with Python projects, but keep C# experience visible
- **3 strong Python AI projects > 10 years C# for AI roles**
- Goal is "working on AI" — language is a tool, not an identity
- Apply to BOTH pure-Python AND hybrid roles. Take first decent offer.

## Path Options Compared

### Faster Path (3-4 months) — High Risk
- Quit .NET job → full-time learning
- Cuts timeline in half
- Requires 6+ months savings
- Best for: young, no dependents, strong runway

### Smartest Path (2-Stage) — Recommended
- Stage 1: Take hybrid .NET+AI role (months 0-6)
- Stage 2: Pivot to pure Python AI role (months 6-18)
- Income continuity + real AI experience
- Best for: most people, especially with responsibilities

### Side-Project Path — Low Risk, Slow
- Stay in .NET job, learn Python evenings/weekends
- Ship 2-3 projects in 6-9 months
- Jump directly to Python AI role
- Best for: high discipline, no urgency
